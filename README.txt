# Movie Recommendation System

This is a simple content-based movie recommender system using TF-IDF and cosine similarity.

## How to Run

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Download the `movies.csv` dataset from Kaggle (TMDB 5000 Movie Dataset or similar):
   https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

3. Place the `movies.csv` in the project folder.

4. Run the program:
   ```
   python main.py
   ```

Enter a movie title and see the top 5 similar movies recommended.
